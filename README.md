🛰 Landsat L1 Image Processing Pipeline
End-to-end radiometric and atmospheric correction pipeline for Landsat 8/9 OLI data.

radiative transfer model. Outputs analysis-ready GeoTIFFs with NDVI.
Tools: Python, rasterio, Py6S, GDAL, NumPy
